# engraph v2.2b — MCP Server Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `engraph serve` command that runs an MCP stdio server exposing 7 read-only vault tools (search, read, list, vault_map, who, project, context) via the official Rust MCP SDK.

**Architecture:** New `src/serve.rs` module with `EngraphServer` struct implementing rmcp's `ServerHandler` via `#[tool_router]` macro. All fields wrapped in `Arc` (+ `tokio::sync::Mutex` for Store/Embedder) to satisfy rmcp's `Clone` requirement. Tool handlers call existing context/search functions synchronously. Main function becomes `#[tokio::main]` async.

**Tech Stack:** Rust, rmcp 1.2 (official MCP SDK), tokio, schemars

**Spec:** `docs/specs/2026-03-24-engraph-v2.2b-design.md`

- [ ] **Step 3: Implement tool handlers with `#[tool_router]`**

```rust
#[tool_router]
impl EngraphServer {
    #[tool(description = "Hybrid search across the vault using semantic, keyword, and graph lanes. Returns ranked results with scores, file paths, headings, snippets, and docids.")]
    async fn search(&self, #[tool(param)] params: SearchParams) -> Result<CallToolResult, McpError> {
        let top_n = params.top_n.unwrap_or(5);
        let store = self.store.lock().await;
        let mut embedder = self.embedder.lock().await;
        let output = search::search_internal(&params.query, top_n, &store, &mut embedder, &self.hnsw_index)
            .map_err(|e| mcp_err(&e))?;
        to_json_result(&output.results)
    }

    #[tool(description = "Read a note's full content with metadata, frontmatter, tags, and graph connections (wikilinks, mentions).")]
    async fn read(&self, #[tool(param)] params: ReadParams) -> Result<CallToolResult, McpError> {
        let store = self.store.lock().await;
        let ctx = self.make_context_params(&store);
        let note = context::context_read(&ctx, &params.file).map_err(|e| mcp_err(&e))?;
        to_json_result(&note)
    }

    #[tool(description = "List notes filtered by folder path and/or tags. Returns paths, docids, tags, and edge counts.")]
    async fn list(&self, #[tool(param)] params: ListParams) -> Result<CallToolResult, McpError> {
        let store = self.store.lock().await;
        let ctx = self.make_context_params(&store);
        let tags = params.tags.unwrap_or_default();
        let limit = params.limit.unwrap_or(20);
        let items = context::context_list(&ctx, params.folder.as_deref(), &tags, limit)
            .map_err(|e| mcp_err(&e))?;
        to_json_result(&items)
    }

    #[tool(description = "Vault structure overview: folders with note counts, top tags, recent files, total stats. Use this to orient yourself in a new session.")]
    async fn vault_map(&self) -> Result<CallToolResult, McpError> {
        let store = self.store.lock().await;
        let ctx = self.make_context_params(&store);
        let map = context::vault_map(&ctx).map_err(|e| mcp_err(&e))?;
        to_json_result(&map)
    }

    #[tool(description = "Person context bundle: full person note, vault mentions with snippets, wikilink connections. Use to understand who someone is and their recent activity.")]
    async fn who(&self, #[tool(param)] params: WhoParams) -> Result<CallToolResult, McpError> {
        let store = self.store.lock().await;
        let ctx = self.make_context_params(&store);
        let person = context::context_who(&ctx, &params.name).map_err(|e| mcp_err(&e))?;
        to_json_result(&person)
    }

    #[tool(description = "Project context bundle: project note, child notes, active tasks (unchecked checkboxes), team members, recent daily mentions.")]
    async fn project(&self, #[tool(param)] params: ProjectParams) -> Result<CallToolResult, McpError> {
        let store = self.store.lock().await;
        let ctx = self.make_context_params(&store);
        let proj = context::context_project(&ctx, &params.name).map_err(|e| mcp_err(&e))?;
        to_json_result(&proj)
    }

    #[tool(description = "Rich topic context bundle: runs hybrid search, reads full note content, expands via graph, assembles within character budget. Use for deep context on any topic.")]
    async fn context(&self, #[tool(param)] params: ContextToolParams) -> Result<CallToolResult, McpError> {
        let store = self.store.lock().await;
        let mut embedder = self.embedder.lock().await;
        let ctx = self.make_context_params(&store);
        let budget = params.budget.unwrap_or(32000);
        let bundle = context::context_topic_with_search(&ctx, &params.topic, budget, &mut embedder, &self.hnsw_index)
            .map_err(|e| mcp_err(&e))?;
        to_json_result(&bundle)
    }
}
```

impl EngraphServer {
    fn make_context_params<'a>(&'a self, store: &'a Store) -> ContextParams<'a> {
        ContextParams {
            store,
            vault_path: &self.vault_path,
            profile: self.profile.as_ref().as_ref(),
        }
    }
}

fn mcp_err(e: &anyhow::Error) -> McpError {
    McpError::new(rmcp::model::ErrorCode::INTERNAL_ERROR, format!("{e:#}"), None::<()>)
}

fn to_json_result<T: serde::Serialize>(value: &T) -> Result<CallToolResult, McpError> {
    let json = serde_json::to_string_pretty(value)
        .map_err(|e| McpError::new(rmcp::model::ErrorCode::INTERNAL_ERROR, e.to_string(), None::<()>))?;
    Ok(CallToolResult::success(vec![Content::text(json)]))
}

#[tool_handler]
impl ServerHandler for EngraphServer {
    fn get_info(&self) -> ServerInfo {
        ServerInfo {
            instructions: Some("engraph is a vault intelligence tool for Obsidian vaults. Use vault_map to orient yourself, search to find notes, read to get full content, who/project for context bundles.".into()),
            capabilities: ServerCapabilities::builder().enable_tools().build(),
            ..Default::default()
        }
    }
}

- [ ] **Step 5: Implement `run_serve`**

```rust
pub async fn run_serve(data_dir: &Path) -> Result<()> {
    let db_path = data_dir.join("engraph.db");
    let store = Store::open(&db_path).context("opening store")?;

    let models_dir = data_dir.join("models");
    let embedder = Embedder::new(&models_dir).context("loading embedder")?;

    let hnsw_dir = data_dir.join("hnsw");
    let hnsw_index = HnswIndex::load(&hnsw_dir).context("loading HNSW index")?;

    let vault_path_str = store.get_meta("vault_path")?
        .ok_or_else(|| anyhow::anyhow!("No vault path in index"))?;
    let vault_path = PathBuf::from(&vault_path_str);

    let profile = Config::load_vault_profile().ok().flatten();

    let server = EngraphServer {
        store: Arc::new(Mutex::new(store)),
        embedder: Arc::new(Mutex::new(embedder)),
        hnsw_index: Arc::new(hnsw_index),
        vault_path: Arc::new(vault_path),
        profile: Arc::new(profile),
    };

    eprintln!("engraph MCP server starting...");

    let transport = rmcp::transport::io::stdio();
    let server_handle = server.serve(transport).await
        .context("MCP server failed to start")?;

    server_handle.waiting().await?;

    Ok(())
}
```
